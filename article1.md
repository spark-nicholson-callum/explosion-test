最低限の流体力学
----------------

### Navier-Stokes

流体力学っとは、流れる物の動きの研究です。
みんなが学校で勉強した物体の動きのニュートン力学と違って、団体で動く物に関しての領域です。

ニュートンの場合は有名の式で物の動きを計算できます
```
$$r
\bold{F} = m \bold{a}
$$
```
ですが、力学で一番ほしい情報は速度の変更です。微積分で書き直して、
```
$$
\frac{d \bold{v}}{d t} = \frac{1}{m} \bold{F}
$$
```
になります。

流体力学には同じ役目の式がもちろんあります
```
$$
\frac{D \bold{u}}{D t} = \nu \nabla^2 \bold{u} - \frac{1}{\rho}\nabla p + \frac{1}{\rho}\bold{f}
$$
```
ニュートンに比べたらかなり複雑ですが、形が同じです

実はこれは非圧縮性のケースです、圧力が変わっても大きさが変わらないってことです。
気体でもこれは意外と大丈夫です、箱の中やマッハ程度の速度の場合はやっぱりだめですが、一般の場合が関わらないほうがいいですよ。
爆発がほしいならさすがにこれはアウトですが、完全なシミュレーションが必要にありません。
まず非圧縮性の概算をして、非物理学な爆発っぽい見た目を追加するって感じで問題ないです。

項を1個ずつ説明します。

* $\frac{D \bold{u}}{D t}$: これは速度の導関数、流体の場合は$\bold{v}$ではなく$\bold{u}$を使います、あと導関数は$d$ではなく$D$で書く、これについては後ほど説明します。
* $\nu \nabla^2 \bold{u}$: 粘性からの力です、流体自体が動きを抵抗する力ですね。気体の場合は$\nu$がほとんど$0$ので今回これは無視して大丈夫です。
* $\frac{1}{\rho}\nabla p$: 圧力からの力です。流体の一部が周りの流体を押す力ですね。非圧縮性でもちゃんとあります。
* $\frac{1}{\rho}\bold{f}$: 外部の力です。流体の場合は質量の$m$ではなく、質量密度の$\rho$を使いますが、ニュートンと同じです。

ニュートンに比べると、流体の中からの力がありますが、まぁ同じ見たいな式です。

### 移流（Dの意思）

導関数では$d$ではなく$D$を使いましたが、ちゃんと意味があります。
これは物質微分っていうものです。

流体では速度を変更するのは力だけではなく、流れているので粒子自体が動いています。
ほしいのはある粒子の速度ではなく、特定の場所にある粒子の速度です。
この粒子が速度（と他のパラメータ、温度とか）を運んで別の場所に移動するのは移流っと言います。

物質微分がある粒子（流体要素）の変更って意味です。
シミュレーションのために特定の場所の速度の変更がほしいです。
こちらの式で計算できます

```
$$
\frac{D \bold{u}}{D t} = \frac{\partial \bold{u}}{\partial t} + \( \bold{u} \cdot \nabla \)\bold{u}
$$
```

また、項ごと説明します

* $\frac{\partial \bold{u}}{\partial t}$: 特定の場所の速度の変更、計算したいのはこれです。
* $\( \bold{u} \cdot \nabla \)\bold{u}$: すごい書き方ですが、これは速度の移流です。流体のむずかしさが大体ここです。

移流はまぁ、計算ではなくシミュレーションしますので気にしなくもて大丈夫ですが、調べたい人はこの3文字の分だけは奥がかなり深い。

### 発散と回転

ベクトル場の導関数が種類多いです。
発散と回転の計算が必要になりますので、短めに説明します。

#### 発散
```
$$
\nabla \cdot \bold{F}
$$
```
ベクトルがある場所に集めっていく（+ve 発散）か離れていく (-ve 発散）。
発散が丁度0の場合は単純に流れているイメージです。徳に今回は非圧縮性のシミュレーションので発散が0のはずです。

#### 回転
```
$$
\nabla \times \bold{F}
$$
```
ベクトルがねじりこむかの計算です。これはベクトルです。
物理学の授業で学んだ（はず）の右手の法則でねじる方法がわかります。
このねじりの動きがすごく流体っぽいのでちゃんと表示したいですね。

### 圧力の計算

非圧縮性の場合は発散をゼロにするのは圧力です。
動きがある場所に集中ようになったら、その場所の圧力が上がって、発散がゼロの状態に続くの感じです。
この条件を守る圧力がポアソン方程式で計算します

```
$$
\nabla^2 p = \nabla \cdot \bold{p}
$$
```

これは一発で計算できないやつので、何回も反復して圧力に近づくしかないです。
FPSが死ぬのは大体ここです。。。

### まとめて

流体シミュレーションのために空間をボクセルに分けて、各場所に$\frac{\partial \bold{u}}{\partial t}$を計算します。
これを計算するには Navier-Stokes を以下のように利用します

```
$$
\frac{\partial \bold{u}}{\partial t} = - \( \bold{u} \cdot \nabla \)\bold{u} - \frac{1}{\rho}\nabla p + \frac{1}{\rho}\bold{f}
$$
```

言葉にするとこれは
```
[速度の変更] = -[移流] - [圧力の力] + [外部の力]
```

圧力のほうがポアソン方程式で計算する
```
$$
\nabla^2 p = \nabla \cdot \bold{p}
$$
```
